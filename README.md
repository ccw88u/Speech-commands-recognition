# Speech-commands-recognition

Three-word classification
"Yes", "No" and "Marvin"

2 versions:
  - sr.progr.py
  - SR-LSTM.py

Features: Zero Crossing Rate, MFCC, Spectral Centroid, Spectral Rolloff

Neural network: LSTM
